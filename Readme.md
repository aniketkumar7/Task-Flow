## TaskFlow

Minimal task tracker (single-page, fast, no accounts).

### Features

- **Quick add**: type a task, press Enter
- **Edit / delete / mark done**
- **Search + filters**: All / Active / Done
- **Sort**: Newest / Oldest / Title
- **Local-first**: saves to localStorage (auto-migrates from older TaskFlow storage)
- **Light/dark**: theme toggle (respects system by default)

### Run

- Open `index.html` in your browser.

### Keyboard

- **/** focuses Search
- **n** focuses Add task
